Browse Texts

This tool is designed to help you to overview existing texts and to search them according to specific criteria. You have two options.

List un-Translated Texts

Simplified search - this tool which is designed for one sole, specific, and obvious purpose. You can choose the language, set, and a number of displayed results and you will get a list of texts which have no translation yet. You can use the open in a new window command, to save yourself from querying after every translation.

List&Reserve Texts

Text Lister allows you to set very specific criteria for the selected texts. This tool is mainly used when you would like to reserve some texts for yourself, to prevent other authors working with them. You need to select the set, language, and expiration date and add selection criteria to get the list of texts with checkboxes for further reservation. Or you can simply display all texts you have currently reserved (at the bottom).
