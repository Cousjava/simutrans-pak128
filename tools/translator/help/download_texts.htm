Download Texts

This page allows you to get the most up-todate translation file directly usable by your Simutrans game. Simply select your language and choose the version (scenario) you play, switch the output method to file, click Export!, wait a while, and click save file/target as on the resulting link. Created file should be saved to your whatever_path\simutrans\scenario\text directory.

The file might not work well, if a language maintainer set an incorrect coding for the language. For example Czech uses Latin II - ISO 8859-2. In case of problems, please contact the administrator.
