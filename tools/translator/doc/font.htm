Font Information Format - f_desc

Simutrans is capable of displaying any font that whose character set can be encoded into 8-bit charset or is UTF-8 compliant. Simutrans font is proportional. Simutrans uses fonts that are stored in own application specific format (.fnt) or .bdf format. SimuTranslator should not read font information from this file.

SimuTranslator should read the information from font specification files.

The structure of this file is:

  • Line starting with: # is a comment and is not parsed.
  • Word encapsulated in [] is a section header, heder must be alone on the line.
  • Section [Constants] defines the properties of the text window in Simutrans. Currently not used! length of short line: ShortLineLen = 126 (in pixels), length of long line: LongLineLen = 202 (in pixels) and number of short lines, and number of short lines: ShortLineCount = 6. (First x lines in the window are short and remaining lines are long, the number of long lines is not limited.)
  • Section [Letters] defines the width of each character in Simutrans. The syntax is: “character” = width (example: a = 4) - first char is the character itself, then follows space, then equal sign, then space and then char. Width is given in pixels. Space is marked as empty space.

This information file should use UTF-8 encoding. This information will be stored in the attribute f_desc of entity Language.
