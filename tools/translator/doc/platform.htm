Resource Requirements

General Requirements

  • Client - Requirements:
    Simutrans runs on Windows, Linux and BeOS operating system. SimuTranslator should be accessible from at least those systems. Web based applications are ideal for this purpose, because they can be accessed from large variety of operating systems and even mobile devices. SimuTranslator should be designed in such way, that html code will be compatible with newer versions of all mayor browsers (MSIE and Gecko based browsers). There should be no further requirements on client computer.
  • Server - Requirements:
    Simutrans is freeware project, it is not backed up by any resources or finances, so it is important, that it does not require expensive 3rd party software products to operate.
    This does not mean that SimuTranslator server must run on free operating system, but it means that it must not really on features offered only by commercial 3rd party applications.
    Currently all Simutrans related websites are hosted on server donated by Nullus Anxietas, an US based web hosting company. SimuTranslator should be able to run on the server provided by this company. Their server is running Linux with kernel version 2.4.20-31.9 .
    Same requirement hold for any other applications required to run SimuTranslator. Currently SimuTranslator need web server supporting PHP and MySQL database.
